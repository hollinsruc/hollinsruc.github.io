Ubuntu扁平化主题——Flatabulous

发表于 2016-04-03 | 分类于 Linux | 阅读次数
作为初学者，我一度对Ubuntu其貌不扬的界面感到不是。直到有一天，我发现了一款精美的系统主题——Flatabulous。


笔者前两天在重装Ubuntu的时候，无意中发现了这款精美的主题。于是，毫不吝啬地推荐给大家。

Flatabulous

这是一款为Ubuntu等Linux系统制作的扁平化主题，主要基于Ultra-Flat主题。

安装

这款主题是专门为Ubuntu上unity环境设计的，而且笔者已经在个人的电脑上进行了测试，表现出色。当然，可能在其他版本的Linux系统中也可以运行。

终端

原作者提到要安装Ubuntu tweak tool

To install the theme, you must first install the Ubuntu tweak tool

值得注意的是，Ubuntu tweak tool是为Ubuntu 13.10及以前的版本准备的。

该项目已托管在Github上，可以前往项目主页或者点击此处下载主题资源。

然后将文件解压并且移动到/usr/share/themes/ 目录下

扁平图标


在图标方面，我使用ultra-flat-icons主题。它包括了蓝色(推荐)，橙色和薄荷绿三种颜色。您可以在终端中输入一下命令已安装它。

使用主题

现在按下Super键，进入Dashboard，搜索Ubuntu Tweak (或者Unity Tweak tool)并运行该程序。

以Unity Tweak Tool 为例，在外观选项卡中有一个”主题”选项，选择”Flatabulous-master主题。在”图标”选项中，选择”Ultra-flat“图标。
